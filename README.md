
# Frontend (web UI) connection with Python Backend via FLASK

This repo contains a very simple and short project of frontend connection with backend via flask. This project contains:
1. templates folder -> index.html our UI file.
2. app.py file, which is basically our connection between our UI and Python backend logic.
3. logic.py, this files contails a function name messege_printer(param), which basically is our message which we want to display.


## Installation

Installing flask

```bash
  pip install flask
```
